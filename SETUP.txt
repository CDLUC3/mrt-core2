
* Requirements
JDK 1.6(+)
ant
maven
________________________________
* Download core
hg clone https://bitbucket.org/merritt/mrt-core
________________________________

* Build core project using maven on mrt-core
cd mrt-core
mvn clean
mvn install
________________________________

* Maven release

mvn --batch-mode -DreleaseVersion=2019.1.alpha -DdevelopmentVersion=2.0-SNAPSHOT -Dtag=2019.1.alpha release:prepare
